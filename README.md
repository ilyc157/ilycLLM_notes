## Git基础
### 基础命令
- git add .
- git commit -m '描述信息'
- git push
- git pull

### 实战操作
1. fork
2. git clone url
3. git fetch origin #clone只会拉下来默认分支，通过fetch获取远程的所有分支列表
4. git checkout -b new_branch_name origical_branch_name
5. 完成文件修改
6. git add .
7. git commit -m “描述信息”
8. git push [远程仓库名] [本地分支名]:[远程分支名]
9. 创建pull request，贡献原repo
